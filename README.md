# ピタゴラスイッチゲーム

NHK教育番組「ピタゴラスイッチ」をモチーフにした3D物理シミュレーションゲームです。
3次元空間でビー玉を落とし、ドミノを倒したり、障害物を配置して遊ぶことができます。

## 機能

- 🎯 **ビー玉の落下**: リアルな物理演算によるビー玉の落下
- 🎴 **ドミノ**: 倒れるドミノを配置
- 📐 **坂**: ビー玉が転がる坂を配置
- 📦 **箱**: 障害物として箱を配置
- 🎮 **インタラクティブ操作**: マウスでカメラを操作、オブジェクトをクリックして削除

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで `http://localhost:3000` が自動的に開きます。

## 操作方法

### カメラ操作
- **マウスドラッグ**: カメラを回転
- **マウスホイール**: ズームイン/アウト

### ゲーム操作
- **ビー玉を落とす**: 上部のボタンからビー玉を落下させます
- **ドミノを追加**: ランダムな位置にドミノを配置します
- **坂を追加**: ビー玉が転がる坂を配置します
- **箱を追加**: 障害物として箱を配置します
- **オブジェクト削除**: オブジェクトをクリックすると削除されます（ビー玉以外）
- **すべてクリア**: すべてのオブジェクトを削除します
- **ビー玉をリセット**: ビー玉を初期位置に戻します

## 技術スタック

- **Three.js**: 3Dグラフィックスライブラリ
- **Cannon-es**: 物理エンジン
- **TypeScript**: 型安全な開発
- **Vite**: 高速なビルドツール

## ビルド

```bash
npm run build
```

ビルドされたファイルは `dist` ディレクトリに出力されます。
